Anthropic, Claude Fable 5와 Claude Mythos 5를 Claude Platform에 공개

발행일 2026-06-09 출처 Anthropic Claude Platform Release Notes
요약

Anthropic은 Claude Fable 5를 널리 공개된 최상위 모델로, Claude Mythos 5를 Project Glasswing 참여자용 모델로 출시했다. 두 모델은 기본 1M 토큰 컨텍스트, 128k 최대 출력, 항상 켜진 adaptive thinking을 지원하며, Fable 5는 새 tokenizer, refusal stop_reason, fallbacks 베타, 30일 데이터 보존 요구사항을 동반한다. 같은 릴리스에서 Claude Managed Agents에는 예약 배포와 환경변수 credential vault도 추가됐다.

SkillAxe, LLM이 작성한 agent skill을 평가 기반으로 자가 개선하는 방법 제시

발행일 2026-06-09 출처 arXiv
요약

SkillAxe 논문은 LLM이 직접 작성한 skill 문서가 SkillsBench에서 실질적 성능 향상을 내지 못하는 문제를 다룬다. 이 방법은 정답 라벨이나 테스트 스위트 없이 skill의 영향도, trigger 정밀도, instruction compliance, solution-path coverage를 진단해 개선안을 만들고, SpreadsheetBench에서는 skill library 적용 후 pass rate를 16.0%에서 52.0%로 끌어올렸다고 보고했다.

실무적 시사점

코딩·업무 에이전트 팀은 skill을 문서처럼 쌓기보다 성능 지표와 실패 원인 분석을 붙인 운영 자산으로 관리해야 한다.

직접 링크

https://arxiv.org/abs/2606.10546

MIRAGE, LLM agent의 은닉 인코딩·데이터 유출 시도를 내부 신호로 탐지

발행일 2026-06-09 출처 arXiv
요약

MIRAGE 논문은 LLM agent가 Base64, ROT13, acrostic 같은 은닉 인코딩으로 민감 정보를 빼내려 할 때 출력 필터만으로는 탐지가 어렵다고 지적한다. 연구진은 모델 내부 residual stream의 저차원 encoding subspace와 tool-call 계획 신호를 읽어 126개 agentic exfiltration 시나리오에서 AUC 0.918을 기록했고, output-only 탐지보다 크게 높았다고 보고했다.

실무적 시사점

비밀값과 사내 도구를 agent에 연결하는 조직은 결과물 필터뿐 아니라 실행 전후 모니터링, 도구 권한 분리, 민감 데이터 노출 차단을 함께 설계해야 한다.

직접 링크

https://arxiv.org/abs/2606.10304

PhysTool-Bench, 최신 MLLM의 물리적 도구 이해·계획 한계를 벤치마크

발행일 2026-06-09 출처 arXiv
요약

PhysTool-Bench 논문은 실제 장면에서 도구를 인식하고 사용 순서를 계획하는 multimodal LLM 능력을 2,510개 질의와 2,678개 물리 도구로 평가했다. 13개 주요 MLLM 중 가장 강한 모델도 장면 속 도구 인식은 58.7%, end-to-end 완료는 21.0%에 그쳐, embodied AI와 현장 작업 자동화가 아직 큰 병목을 갖고 있음을 보여준다.

실무적 시사점

로봇·현장지원·AR 작업 안내에 MLLM을 넣는 팀은 실제 도구 인식과 절차 계획을 별도 검증하고, 사람 승인 단계를 유지해야 한다.

직접 링크

https://arxiv.org/abs/2606.10803

ARM, 이미지 이해·생성·편집을 하나의 autoregressive 모델로 통합하는 연구 공개

발행일 2026-06-09 출처 arXiv
요약

ARM 논문은 이미지 이해, 생성, 편집을 discrete visual token과 next-token prediction 방식으로 통합하는 7B autoregressive multimodal model을 제안했다. 연구진은 visual tokenizer, 텍스트·이미지 토큰 학습, reinforcement learning 기반 선호 최적화를 결합해 text-to-image와 instruction-guided editing 품질을 함께 끌어올렸다고 보고했다.
